Front End

Front-end web development is the development of the graphical user interface of a website through the use of HTML, CSS, and JavaScript so users can view and interact with that website.

Back End

The back-end is the code that runs on the server, that receives requests from the clients, and contains the logic to send the appropriate data back to the client. The back-end also includes the database, which will persistently store all of the data for the application.

Database

A database is a collection of data that's organized and stored electronically on a computer system. Databases are designed to store large amounts of information and allow multiple users to access and query the data.
